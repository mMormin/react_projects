## Deployment
### Clone it
```cli
git clone https://github.com/mMormin/react-oclock
git switch "ex:recette (queried project)" 
yarn (to install dependencies)
```

### Install the dependencies
```cli
yarn
```

### Launch it vite
```cli
yarn dev || yarn vite
```
